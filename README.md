# Vayka Portal 🏨✨

[![React](https://img.shields.io/badge/React-18.x-blue.svg)](https://reactjs.org)
[![Vite](https://img.shields.io/badge/Vite-6.x-purple.svg)](https://vitejs.dev)
[![Redux](https://img.shields.io/badge/Redux-Toolkit--Query-green.svg)](https://redux-toolkit.js.org)

The premium user-facing web application for **Vayka**. This portal allows users to discover, book, and manage luxury hotel stays with a seamless and high-performance interface.

---

## 🎨 Design Philosophy

- **Premium Aesthetics**: Clean, modern UI using curated color palettes and Google Fonts (Inter/Outfit).
- **Responsive Mastery**: Fully optimized for Desktop, Tablet, and Mobile devices.
- **Dynamic Interactions**: Smooth animations powered by Framer Motion and modern CSS transitions.
- **Glassmorphism**: Elegant transparency effects for a sophisticated look.

## 🚀 Top Features

- **Advanced Discovery**: Intelligent filtering and map-based search for high-end properties.
- **Real-Time API**: Powered by **Redux Toolkit Query** for lightning-fast data fetching and automatic cache management.
- **Comprehensive Booking Flow**: A secure, step-by-step process for hotel reservations.
- **Partner Dashboard**: Dedicated interface for hotel owners to apply and manage their properties.
- **Visual Excellence**: Professional image galleries with adaptive grid layouts.
- **Notification Center**: Real-time updates on booking status and application approvals.

## 💻 Tech Stack

- **Framework**: React / Vite
- **State Management**: Redux Toolkit Query (RTK Query)
- **Styling**: Vanilla CSS (Custom tokens) & Tailwind CSS
- **Icons**: Lucide React & Heroicons
- **Typography**: Inter (Modern/Clean) & Outfit (Geometric/Premium)


## 🏗️ Project Structure

- `/src/pages`: Main application screens (Home, Profile, PartnerForm, etc.)
- `/src/components`: Reusable UI components (Navbar, Footer, Gallery)
- `/src/services`: API definitions and RTK Query endpoints
- `/src/store`: Redux store configuration and slices
- `/src/styles`: Design system tokens and global CSS
